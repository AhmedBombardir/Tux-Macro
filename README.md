# Tux-Macro
Tux-Macro is the first ever free, open source Bee Swarm Simulator macro dedicated for linux users

Right now it features only gathering with back to hive by reset option, along with some premade patterns.
It supports custom patterns as well as custom paths

Imporant info:
- Set up a global hotkey to kill the macro, for example f3
- The macro is in prerelease state, it may contain bugs / not work properly
- report issues on discord

Dependencies needed:
- subprocess
- threading
- time
- os
- re
- pygame
- opencv
- numpy
- evdev
